
# Disney App

Small application to view Disney content by categories

Use
- React
- Next js
- Tailwind
- TypeScript
- Context API
- Firebase

## Installation

Install my-project with npm

```bash
  npm install disneyapp
  cd disneyapp
  npm i
  npm run dev
```
Open http://localhost:3000 with your browser to see the result.

That's all! 
## Deployment

To deploy this project run

```bash
  npm run build
```
